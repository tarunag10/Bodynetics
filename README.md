# Bodynetics Physiotherapy Clinic

A polished React/Vite portfolio website for Bodynetics, a London physiotherapy clinic brand.

## Pages

- Home
- Services
- Conditions
- About
- Approach
- Fees
- FAQs
- Contact with Calendly placeholder

## Brand System

The website follows the provided Bodynetics brand board:

- Playfair Display for headings and brand accents
- Inter for body copy and UI text
- Deep navy, cobalt blue, powder blue, warm cream, and sunlight yellow
